 Telco Customer Churn Analysis

 Project Overview
This project analyzes customer churn data from a telecom company using Microsoft Excel. The objective is to identify factors affecting customer churn and present insights through Pivot Tables and an interactive Dashboard.

 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard Design

Dataset
The dataset contains information about 7043 telecom customers, including:
- Customer ID
- Contract Type
- Internet Service
- Tech Support
- Churn Status

 Key Performance Indicators (KPIs)
- Total Customers: 7043
- Active Customers: 5174
- Churned Customers: 1869
- Churn Rate: 26.54%
- Retention Rate: 73.46%

 Dashboard Visualizations
1. Churn Distribution
2. Churn by Contract Type
3. Churn by Internet Service
4. Churn by Tech Support

 Key Insights
- Customers with month-to-month contracts have the highest churn rate.
- Fiber optic users show higher churn compared to other internet services.
- Customers without tech support are more likely to churn.
- Most customers are retained, resulting in a retention rate of 73.46%.

 Files Included

 Conclusion
This analysis helps understand customer behavior and highlights factors that influence customer churn. The dashboard provides a clear and interactive way to monitor churn trends and support business decision-making.
